210→## 📝 技术细节
211→
212→### 主题实现
213→- 使用 `document.documentElement.classList.add('dark')` 应用主题
214→- Tailwind CSS 的 `dark:` 前缀自动处理深色模式样式
215→- Zustand persist 中间件持久化主题选择
216→
217→### 快捷键实现
218→- 使用原生 `KeyboardEvent` API
219→- 自动检测输入框状态，避免快捷键冲突
220→- 支持修饰键组合（Ctrl、Shift、Alt、Meta）
221→
222→### 动画实现
223→- 使用 Tailwind 的 `transition-*` 和 `animate-*` 类
224→- 自定义 CSS 动画（`fade-in`、`shimmer`）
225→- 使用 `transform` 而非 `position` 实现动画（性能更好）
