# .agent-hooks.md - Zed Hook 模拟（2026 版）

postSessionHooks:
- auto-test-and-revert: 每次变更后跑 npm run test && typecheck，失败立即 revert + 更新约束
- self-review-loop: PR 前生成视频 + Judge，循环最多 5 次
- danger-approval: rm/migrate/git reset 前暂停问人类
- entropy-cleanup: 每周自动清理

observability: 每次失败截图 + 日志到 docs/exec-plans/
feedbackLoop: 失败自动更新 CLAUDE.md 和 AGENTS.md
